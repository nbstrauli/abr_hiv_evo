This directory contains the code associated with the manuscript: "The genetic interaction between HIV and the antibody repertoire". 

DISCLAIMER:
The code herein is meant as a reference for those that want to see exactly how analyses for this manuscript were carried out, in silico. It is not meant to be code for general use, and will not work if simply imported. That said, there could be useful functions/methods for the experienced python user, who is willing to look.
